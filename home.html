what is Web 3

Web 3.0 is the next third generation of the internet, in which websites and applications will be able to process information in a human-like manner using machine learning (ML), big data, and decentralised ledger technology (DLT), among other technologies. Tim Berners-Lee, the creator of the World Wide Web, initially referred to Web 3.0 as the Semantic Web, with the goal of creating a more autonomous, intelligent, and open internet.
